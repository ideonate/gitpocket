🔑 Token Setup Guide:

Option 1: Fine-grained PAT (Recommended)
1. Go to github.com/settings/tokens/fine-grained
2. Click "Generate new token"
3. Repository access: Choose "Selected repositories" OR "All repositories"
4. Set permissions:
  • Issues: Read and write
  • Pull requests: Read and write
  • Metadata: Read
5. Generate and copy token
⚠️ For Organization Repositories:
• Organization must allow fine-grained PATs
• You may need approval from org owners
• If fine-grained doesn't work, use Classic PAT below
Option 2: Classic PAT (Fallback)
1. Go to github.com/settings/tokens
2. Click "Generate new token (classic)"
3. Select scopes:
  • repo (full control for private repos)
  • OR public_repo (for public repos only)
4. Generate and copy token
